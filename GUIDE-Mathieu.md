# Comment légender vos photos

Un outil simple : vous double-cliquez, vous choisissez un dossier, c'est fini.

---

## Ce dont vous avez besoin

Dans **un seul et même dossier**, mettez ensemble :

- vos photos **.tif**
- votre fichier de légendes **Excel** (`.xlsx`) ou **CSV**

Le fichier de légendes doit avoir **deux colonnes** :

| Filename     | Description                              |
|--------------|------------------------------------------|
| MR10431.TIF  | Contreforts de l'Himalaya… / Himalayan…  |
| MR10432.TIF  | Grey Langur… / Langur gris…              |

C'est exactement le fichier que vous remplissez déjà. Rien à changer.

---

## Les étapes

**1.** Double-cliquez sur **« Légender les photos »**.

**2.** Une fenêtre s'ouvre : choisissez le **dossier** qui contient vos photos
et votre fichier de légendes, puis cliquez sur **Choisir**.

**3.** Patientez quelques secondes. *(La toute première fois seulement,
l'outil installe un petit composant : laissez-le faire, il faut Internet.)*

**4.** Un message vous dit combien de photos ont été légendées. Cliquez sur
**Ouvrir le dossier** pour vérifier. C'est terminé.

La description est maintenant inscrite dans chaque photo : elle apparaîtra
dans le champ **« Légende » / « Description »** de Bridge, Photoshop, et chez
les agences.

---

## La toute première fois : un avertissement de macOS

Comme l'outil ne vient pas de l'App Store, macOS affiche au premier lancement
un message du type *« … n'a pas pu être ouvert car Apple ne peut pas vérifier… »*
ou *« développeur non identifié »*. C'est normal et sans danger. À faire **une
seule fois** :

**1.** Double-cliquez sur **« Légender les photos »**. Le message apparaît :
cliquez sur **OK** (ou Annuler) pour le fermer.

**2.** Ouvrez le menu  → **Réglages Système** → **Confidentialité et sécurité**.

**3.** Descendez tout en bas : une ligne mentionne *« Légender les photos a été
bloqué… »* avec un bouton **« Ouvrir quand même »**. Cliquez dessus.

**4.** Confirmez avec **Ouvrir**, puis saisissez votre **mot de passe** Mac.

Ensuite, le simple double-clic fonctionnera toujours, sans aucun message.

> 💡 **Astuce pour éviter complètement cette étape :** si on vous a remis l'app
> sur une **clé USB** et que vous la copiez depuis la clé vers votre Mac, cet
> avertissement n'apparaît en général pas du tout.

---

## Si quelque chose ne va pas

- **« Aucun fichier Excel ou CSV trouvé »** : votre fichier de légendes n'est
  pas dans le même dossier que les photos. Mettez-les ensemble et recommencez.

- **« Aucune photo ne correspond »** : les noms de la colonne *Filename* ne
  correspondent pas aux fichiers. Vérifiez par exemple que `MR10431` est bien
  écrit pareil des deux côtés (la casse n'a pas d'importance).

- **Le composant ne s'installe pas** : vérifiez votre connexion Internet,
  puis relancez. L'installation n'a lieu qu'une seule fois.
